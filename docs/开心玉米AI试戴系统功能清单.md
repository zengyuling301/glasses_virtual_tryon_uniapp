# 开心玉米 AI 试戴系统功能清单

**版本**：2.0（对齐 2026 年需求对齐会）  
**形态**：原生 APP 内嵌功能模块（非独立小程序 / 独立 APP）  
**技术栈**：前端 uni-app；后端 PHP（ThinkPHP）  
**交付目标**：两个月内完成开发并正式上线（需甲方配合联调与发布）

---

## 总体原则

| 原则 | 说明 |
|------|------|
| 一气呵成 | 取消独立《同意说明书》、引导页及复杂跳转；用户点击入口后**直接进入拍照界面** |
| 先试戴后美颜 | 面宽测算与试戴阶段**禁止美颜/滤镜**；试戴满意后再开启美颜、发型、场景等增强功能 |
| 无参照物拍照 | 现场实时实拍，**无需参照物**；无美颜、无滤镜，采集真实五官、神态与原生脸型比例 |
| 毫米级面宽 | 通过拍照识别面宽，加载态文案「毫米级精准测算中」；总面宽 = **镜片宽度 × 2 + 中梁宽度**（非单一镜片尺寸） |
| 3D 真实试戴 | 平面镜框图自动转 3D 模型试戴，目标还原度 ≥ 98%，接近真实佩戴效果 |
| 商品闭环 | 对接现有商品库，试戴款式可跳转收藏 / 加购 / 下单 |

## 基础能力（商品与 3D 资产）

### 平面图转 3D 建模

- 独立建模管线：将供应商提供的**平面眼镜图**自动转换为 3D 文件格式
- 还原度要求：≥ 98%，减轻平面贴脸违和感
- 与商品库联动：抓取产品详情页**主图**等信息，**自动生成**对应 3D 模型
- 试戴后可跳转至该商品详情链接（收藏、购买）

### 面宽适配算法

- 面部关键点抓取，输出毫米级测算：面宽、脸型比例、五官间距、镜框适配参数
- **适配三级**（列表标签与文案统一）：
  - **精准适配**（品牌蓝）
  - **勉强可用**（中性 / 黄标）
  - **完全不适配**（红标 / 品牌橙趣味引导偏大偏小）
- 列表排序：尺寸 100% 适配优先置顶；再结合脸型、五官、气质排序
- 同款 ±2 尺码推荐（例：测算面宽 53 → 推荐 51～55）

## 步骤一：入口与实拍采集

- ~~启动弹出《同意说明书》，勾选后进入~~ → **已取消**（合规告知并入 APP 既有流程或拍照页轻提示，不单开步骤）
- ~~独立引导页~~ → **已取消**；入口点击后**直接进入拍照界面**
- 现场实时实拍：**无参照物**、无美颜、无滤镜
- 采集真实五官、神态、原生脸型比例（面宽算法与 APP 面宽能力对接）

---

## 步骤二：拍照引导与对齐

- 品牌蓝文案 + 分步动态引导调姿（流程简化，重点优化 **P6～P9** 页面交互）
- 屏幕中央人脸对齐框（蓝 → 达标变绿）
- 实时提示：脸歪、远近、姿态等
- 达标反馈：震动 + 橙点高亮
- 拍照：品牌蓝按钮 + 3 秒倒计时（数字橙色）

---

## 步骤三：AI 测算与智能推荐

- 提交后：品牌蓝环形加载 + 橙点进度，文案「**毫米级精准测算中**」
- 结果卡片：白底蓝框，核心数字品牌蓝加粗（面宽、脸型比例等）
- 联动商品库：按适配三级与排序规则展示推荐列表
- 后台 / 运营侧可查看测算过程与面宽数据（供校准与客服）

---

## 步骤四：选款与虚拟试戴（核心）

- 镜框列表：卡片式展示，每款带适配标签（**绿 / 黄 / 红** 对应三级适配）
- 点击试戴：**3D 模型**合成至真人实拍，保留轮廓与神态（非平面贴图为主路径）
- 多款式切换对比：滑动淡入淡出 + 轻音效
- 试戴文案分档：
  - 精准适配（品牌蓝）
  - 勉强可用（中性）
  - 偏大 / 偏小（品牌橙趣味引导）
- **本步骤不包含美颜**；用户确认试戴效果后再进入步骤五、六

---

## 步骤五：智能美颜（试戴满意后）

- 原则：**先试戴、后美颜**，避免干扰面宽测算精度
- 智能美颜：保留五官特征，优化肤色质感
- 实现：接入自研或**第三方美颜接口**（需评估免费 / 付费方案及大规模用量成本）

---

## 步骤六：发型匹配（试戴满意后）

- 按脸型、镜框风格、气质一键生成 / 替换发型
- 用户可自由切换发型
- 眼镜 + 脸型 + 发型一体预览
- 实现：计划 1.0 接入现有**大模型或第三方服务**，形成完整体验闭环

---

## 步骤七：场景与穿搭（试戴满意后）

- 一键替换实景背景（商务、演讲、峰会、户外、校园、运动等）
- 智能换装（保留脸型、五官、**已选镜框**不变）
- 「一人多场景、一镜多风格」预览
- 实现：同上，大模型 / 第三方接口，与步骤五、六同属增强能力层

---

## 步骤八：收藏、下单与分享

- 一键收藏、加购物车（跳转商品库对应 SKU）
- 下载：有水印版 / 无水印无 logo 版；保存本地（橙按钮 + 蓝橙下载动效）
- 一键分享（含 APP 内朋友圈）
- 「我的」：查看生成记录
- **重新测脸**入口（发型 / 面部变化后二次校准）
